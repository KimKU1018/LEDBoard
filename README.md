# LEDBoard
LED 전광판 앱입니다.

# 사용 기술 정리
### UINavigationViewController
- 기본적으로 Last In First Out

### 화면 전환 방법
- View Controller에서 다른 View Controller를 호출하여 전환
- Navigation Controller를 사용하여 화면 전환
- 화면 전환용 객체 세그웨이(Segueway)를 사용하여 화면 전환
